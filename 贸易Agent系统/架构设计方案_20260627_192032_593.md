## 定位

不是帮人省时间的工具，是替代人力的自动化引擎。

一个人 + 这套系统 = 选品/定价/上架/采购/物流/客服/财务/风控 八个岗位的产出。

你只做两件事：定战略方向，处理系统搞不定的异常。

## 8个Agent职责

| Agent | 做什么 | 自主程度 |
|---|---|---|
| 选品工 | 日扫1688/拼多多，对比eBay/Shopee价差，发现利润>30%的商品 | 全自动入库，定时运行 |
| 定价工 | 逐项算成本（货+运费+佣金+关税+退货预留），给出建议售价 | 全自动，利润达标即定 |
| 上架工 | 生成listing（标题+描述+关键词），自动发布到平台 | 自动发布，文案可调 |
| 采购工 | 检测到订单→检查价格库存→自动向供应商下单 | 涨价≤15%自动下单，超线暂停 |
| 物流工 | 追踪全链路物流→自动通知客户→签收后触发售后跟进 | 全自动 |
| 客服工 | L1物流查询自动回复 / L2质量问题自动生成并发送 / L3纠纷升级 | L1+L2自动，L3通知你 |
| 财务工 | 自动记账→日/周/月利润报表→税务阈值提醒 | 全自动 |
| 风控工 | 每个Agent行动前扫雷：品牌侵权/禁运品/供应商评分/价格异常 | 低中风险自动标记，高风险拦截 |

## 你的操作

**日常**：打开仪表盘看一眼——今日订单数、利润、是否有异常升级。不用动手。

**被通知时**：系统弹出升级（比如供应商断货找不到替代、客户要求退款），你做个选择，系统继续跑。

**定期**：看财务报表，决定要不要调整品类方向或利润率目标。

## 风险控制（三层自动运行）

| 层 | 做什么 | 举例 |
|---|---|---|
| 预防层 | 风控工在每步行动前扫雷 | 品牌词命中→拦截。供应商评分<4.0→拦截 |
